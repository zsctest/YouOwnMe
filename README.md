# YouOwnMe——人情往来APP
#### 已实现功能
软件需求及功能分析
1.	主界面
- 显示当天日期
- 点击日期在下方显示相关记录
- 点击页面悬浮按钮可以添加一条记录
2.	随礼页
- 显示所有随礼记录
- 记录包括收礼人名字、随礼原因、随礼金额和随礼日期
3.	收礼页
- 显示所有收礼记录
- 记录包括送礼人的姓名、收礼原因、收礼金额和收礼日期
